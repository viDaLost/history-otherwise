# История Иначе 3.0

Полная мобильная версия приложения с автоматической генерацией текста и иллюстрации по одной кнопке.

Что появилось:

- Генерация альтернативной истории через Gemini API.
- Автоматическая генерация одной иллюстрации сразу после текста.
- Иллюстрация показывает общее положение и последствия изменения истории.
- Мобильный интерфейс без длинного скролла.
- Вкладки: Обзор, Анализ, Линия, Карта, Иллюстрация, Экспорт.
- Улучшенное заполнение пустых полей.
- История запросов в браузере.

## Какие переменные нужны в Vercel

GEMINI_API_KEY=твой_ключ_Gemini
OPENAI_API_KEY=твой_ключ_OpenAI_для_изображений

## Что заменить

Загрузи все файлы из архива в GitHub с заменой старых:

- package.json
- vercel.json
- api/generate.js
- api/generate-image.js
- public/index.html
- public/styles.css
- public/app.js
- README.md
- .env.example

## Как запустить

1. Загрузи файлы в GitHub.
2. Открой проект в Vercel.
3. Добавь GEMINI_API_KEY.
4. Добавь OPENAI_API_KEY.
5. Нажми Redeploy.
6. Проверь приложение.

## Логика работы

По кнопке "Смоделировать":
1. frontend отправляет запрос в /api/generate
2. получает структурированный исторический ответ
3. сразу отправляет второй запрос в /api/generate-image
4. получает картинку
5. показывает результат и иллюстрацию в одной сессии

## Важно

История в браузере не сохраняет саму картинку, чтобы не перегружать localStorage.
Текстовые результаты сохраняются.
Если ключ OpenAI не добавлен, текст будет работать, а вкладка "Иллюстрация" покажет, что картинка не создана.
