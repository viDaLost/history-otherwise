# История Иначе 2.0

Полная мобильная переработка приложения.

Что изменилось:

- Новый мобильный дизайн.
- Нет длинного полотна результата.
- Результат разделён на вкладки: Обзор, Анализ, Линия, Карта, Экспорт.
- Улучшена форма для телефона.
- Добавлена компактная история запросов.
- Исправлены пустые данные.
- Backend возвращает result, resultText и поля напрямую.
- Добавлены fallback-значения, если Gemini не заполнил часть полей.
- Gemini просит строгий JSON через responseSchema.

## Какие файлы заменить

Загрузи все файлы из архива в GitHub с заменой старых:

- package.json
- vercel.json
- api/generate.js
- public/index.html
- public/styles.css
- public/app.js
- README.md
- .env.example

## Vercel

После загрузки файлов:

1. Открой Vercel.
2. Открой проект.
3. Проверь переменную GEMINI_API_KEY.
4. Открой Deployments.
5. Нажми Redeploy.
6. Проверь сайт.

## Переменная окружения

В Vercel должна быть переменная:

GEMINI_API_KEY=твой_ключ_Gemini

Ключ нельзя вставлять в frontend-файлы.
