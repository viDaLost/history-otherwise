# История Иначе

Веб-приложение на русском языке для генерации альтернативной истории через OpenAI API.

## Что умеет

- Создание альтернативного исторического сценария.
- Защищённый серверный маршрут `/api/generate`.
- OpenAI API ключ хранится только на сервере.
- Красивый вывод результата по блокам.
- Временная линия событий.
- Карта причин и последствий.
- История запросов в localStorage.
- Экспорт в `.txt` и `.json`.
- Краткая версия, версия для YouTube и версия для озвучки.
