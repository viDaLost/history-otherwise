# Иная История 4.0

Версия без генерации изображений.

Что изменилось:

- Вместо крутящейся загрузки добавлена полоска прогресса.
- Показываются проценты.
- Показывается примерное оставшееся время.
- Ответы стали конкретнее.
- Backend просит Gemini давать годы, участников, действия, последствия и бытовые изменения.
- Усилен fallback, чтобы результат не становился пустым или расплывчатым.
- Название приложения обновлено на «Иная История».

Важно:

Процент загрузки считается по прошедшему времени и средней длительности запроса.
Точный прогресс Gemini API получить нельзя без streaming-архитектуры, но шкала доходит до 100% только после получения ответа.

## Какие файлы заменить

Загрузи все файлы из архива в GitHub с заменой старых:

- package.json
- vercel.json
- api/generate.js
- public/index.html
- public/styles.css
- public/app.js
- README.md
- .env.example

## Vercel

После загрузки:

1. Открой Vercel.
2. Открой проект.
3. Проверь GEMINI_API_KEY.
4. Сделай Redeploy.
5. В Telegram закрой WebApp и открой его заново.
